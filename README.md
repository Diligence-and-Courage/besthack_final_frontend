# React Boilerplate
React boilerplate project with TS, Styled Components, Tailwind with SA(C)SS support

After cloning project go to project directory and do:

```
npm install
```

# Start Development

```
npm run start
```

# Build Production

```
npm run build
```

# Lint

```
npm run lint
```

# Fix Linter Issues

```
npm run lint:fix
```
